# Embedded IoT Platform for Intelligent Living Space

Secured First Prize – Spectrum 2K26
Published Research Paper – ICETIS 2026

## Overview

Embedded IoT Platform for Intelligent Living Space is a distributed ESP32-based smart living space designed to improve safety, security, energy efficiency, and remote monitoring.

The system uses multiple ESP32 nodes to monitor environmental parameters and perform intelligent local decision-making while providing remote access through IoT connectivity.

## Features

- Smart Lighting Control
- Gas Leak Detection
- Motion Detection
- Temperature & Humidity Monitoring
- Automatic Exhaust Fan Control
- Smart Window Control
- Soil Moisture Monitoring
- Automatic Irrigation
- Water Tank Monitoring
- ESP32-CAM Security Module
- Mobile Dashboard Monitoring

## Hardware Used

- ESP32
- ESP32-CAM
- DHT11
- MQ2 Gas Sensor
- PIR Sensor
- LDR Sensor
- Ultrasonic Sensor
- Soil Moisture Sensor
- Servo Motors
- Relay Modules
- OLED Display

## Software Used

- Embedded C
- Arduino IDE
- RemoteXY

## Project Architecture

The system consists of multiple distributed ESP32 nodes:

### Node 1
- Temperature Monitoring
- Humidity Monitoring
- Gas Detection
- Motion Detection
- Smart Lighting
- Exhaust Control

### Node 2
- Water Tank Monitoring
- Soil Moisture Monitoring
- Irrigation Control
- Rain Detection

### Security Node
- ESP32-CAM
- Door Access Control
